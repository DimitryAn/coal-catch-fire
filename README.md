# Запуск сервиса 
Для запуска нашего web-сервиса необходимо проделать следующие шаги.
1) У вас должен быть установлен Docker, в случае отсутствия можно скачать с офицального сайта по ссылке - https://www.docker.com/get-started/
2) Необходимо склонировать проект с данного репозитория. Допустим, по https
```bash
git clone https://github.com/DimitryAn/coal-catch-fire.git
```
3) После загрузки проекта запускаем с помощью команды
```bash
docker compose up --build
```
4) Чтобы проверить работу, необходимо открыть страницу  http://localhost:8000/ в вашем браузере 
5) Для завершения работы приложения выполните в термениле комбинацию 
```bash
Ctrl + C
```
5) Завершение работы всех контейнеров
```bash
docker compose down
```
